# Northpaw

Northpaw is a pet care platform for organizing pet profiles, health timelines, records, insurance details, and future care-cost insights.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase Auth and Postgres
- Vercel

## Features

- User signup and login
- Pet profile creation and management
- Health timeline events
- Current weight derived from timeline records
- Supabase-backed data model with owner-scoped access
